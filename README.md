# Zookeepr


## Purpose
This program demonstrates a web server using the Express.js framework and adds it to a front-end application called Zoo Keepr. The site’s data will go live on the server you build so that anyone can access it without having to download it. This application will also be deployed to Heroku, a cloud application platform that allows you to deploy applications with servers to a webpage.
